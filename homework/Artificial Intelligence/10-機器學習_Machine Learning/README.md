# 機器學習
## 馬可夫鍊模型
- 從目前狀態轉移到下一個狀態，狀態影響機率
## 隱馬可夫鍊模型
- 有一狀態序列是無法觀察到的，但可以觀察到表象序列，利用表象序列回推狀態序列
## 維特比演算法
- 尋找產生某表現序列的最可能隱狀態序列\
[維特比演算法](https://zh.wikipedia.org/wiki/%E7%BB%B4%E7%89%B9%E6%AF%94%E7%AE%97%E6%B3%95)
